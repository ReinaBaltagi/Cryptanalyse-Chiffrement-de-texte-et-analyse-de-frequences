# Cryptanalyse-Chiffrement-de-texte-et-analyse-de-frequences
Ce projet s'agit d'un travail sur le chiffrement monoalphabétique, à savoir du chiffrement par permutation et d'une analyse des fréquences pour le déchiffrer sans utiliser la permutation secrète trouvée. 
le texte utilisé est le suivant:  https://ecogestion.univ-paris13.fr/parcours-big-data-analyse-et-business-intelligence/
